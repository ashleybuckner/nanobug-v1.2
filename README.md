# nanobug-v1.2

This is a *very* simple monitor for the 6809 microprocessor

## Hardware

A 6850 ACIA at 0xbf80
Ram from 0x0000

## Commands

Dnnnn		Dump memory in hex format (q to quit, space for next page)
Gnnnn		Run programme form location
L			Load S19 from console
Mnnnn		Modify memory
R			Dispay registers
